# Radio Station Collection

This repository contains a curated collection of radio stations in M3U playlist format. The `stations.m3u` file includes various internet radio stations from around the world, featuring diverse genres and themes.

## Contents

- `stations.m3u`: The main M3U playlist file containing radio station entries.
- `icons/`: Directory containing logo images for each radio station.

## How to Use

1. Download or clone this repository.
2. Open the `stations.m3u` file in a media player that supports M3U playlists (e.g., VLC Media Player, iTunes, or compatible streaming apps).
3. The playlist will load all the radio stations, and you can select and play them directly.
